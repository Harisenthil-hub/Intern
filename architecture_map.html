3. The Conditional Child Nodes

TankTable.jsx

When: viewMode === 'list'

Inputs: Receives tanksList array.

Mounts: HTML standard <table> to render rows. Has a button to trigger 'Add' or 'View'.

AddTankPage.jsx

When: viewMode === 'add' || 'edit'

Feature: If adding, local form state starts empty. If editing, receives initialData prop and fills inputs.

Mounts: Two-column grid using external standalone Input form children.

ViewTankPage.jsx

When: viewMode === 'view'

Feature: Read-only view. Checks the _mode: 'post' flag.

Mounts: Simple locked view details. If not posted, displays the "Edit Tank" button.

Crucial Design Pattern: Inline Form Extractions

In the Add pages, <Input> and <Select> wrappers are defined OUTSIDE the main component lifecycle function (e.g., function F() { ... } defined globally). This prevents React from generating a new function reference during every local re-render (keystroke), solving a critical DOM unmounting bug where input fields lost focus.
